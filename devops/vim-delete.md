## 全部删除

### 方法一：gg + dG（推荐）
```
gg      # 跳转光标到文件开头
dG      # 从当前位置删除到文件末尾
```

### 方法二：ggvG
```
gg      # 跳转光标到文件开头
v       # 进入可视模式
G       # 选中到文件末尾
d       # 删除选中内容
```

### 方法三：:%d
```
:%d     # 冒号进入命令模式，% 代表全部行，d 代表删除
```

## 整行删除

| 命令 | 说明 |
|------|------|
| `dd` | 删除光标所在整行 |
| `ndd` | 删除光标所在行及后续 n 行（如 `5dd` 删除 5 行） |
| `d` + `n` + `j` | 删除当前行及下 n 行 |

---

## 删除光标位置

| 命令 | 说明 |
|------|------|
| `x` | 删除光标所在字符 |
| `X` | 删除光标前一个字符 |
| `dw` | 删除光标到单词末尾 |
| `de` | 删除光标到单词末尾（包含空格） |
| `d0` | 删除光标到行首 |
| `D` 或 `d$` | 删除光标到行尾 |
| `d^` | 删除光标到行首（不含空格） |

## 删除技巧

### 组合快捷键说明

| 按键 | 作用 |
|------|------|
| `gg` | 跳转到文件开头 |
| `G` | 跳转到文件末尾 |
| `v` | 进入可视模式 |
| `d` | 删除（配合 motion 使用） |
| `%` | 代表全部行 |
| `0` | 移动到行首 |
| `$` | 移动到行尾 |

### 常用组合

- **删除当前行**： `dd`
- **删除 3 行**：`3dd`
- **删除光标后所有内容**：`dG`
- **删除光标前所有内容**：`dgg`
- **删除单词**：`dw`（光标在单词开头）或 `de`（光标在单词任意位置）

### 撤销操作

- `u` - 撤销上一次操作
- `Ctrl + r` - 重做（撤销撤销）
